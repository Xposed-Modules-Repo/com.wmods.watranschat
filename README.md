# WhatsApp消息自动翻译 外贸必备工具WATransChat

在 WhatsApp 中深度集成实时自动翻译功能，输入框可直接使用中文发送消息，对方将自动收到对应的本地语言内容。聊天记录中的任何语言消息也会实时翻译为中文，让跨语言沟通更加高效顺畅。

## 功能特点

- 支持输入中文直接发送，自动翻译为对方当地语言
- 自动翻译聊天记录中的任何语言消息为中文
- 支持 8 种翻译接口
- 支持 248 种语言
- 自动识别并显示：
  - 对方国家
  - 对方语言
  - 当地货币
  - 当地时间
- 支持自定义译文颜色

## 使用优势

帮助您快速了解客户背景，提升沟通效率与专业度，让全球客户沟通更加轻松便捷。


## 安装说明

### Root / LSPosed 版本

1. 请确保您的设备已获得 Root 权限  
2. 在你的设备上安装 LSPosed（Zygisk 或 Riru 版本）。
3. 下载并安装 WATransChat APK
4. 打开 LSPosed Manager 应用
5. 启用 WATransChat 模块。
6. 确保 WhatsApp 被选中
7. 强制停止 WhatsApp
8. 打开 WhatsApp，确认模块是否激活。

> 推荐使用最新版本的 LSPosed，以获得更好的兼容性与稳定性。

---

### 无需 Root 版本

如果您的设备没有 Root 权限，也可以使用无需 Root 的版本，请访问官网https://zla5.github.io
